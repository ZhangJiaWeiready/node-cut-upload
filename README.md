#### 使用方法：
+ `npm install`安装所有的依赖
+ `npm run start`启动服务，并自动打开浏览器
#### 注意:*node的版本要>=7.6*, 因为里面使用了async和await语法，目前node的LTS版本已经是8.9.0，所以请各位放心使用
#### 详细的原理和步骤，请参见文章：[Node+H5实现大文件分片上传](https://segmentfault.com/a/1190000008899001)
